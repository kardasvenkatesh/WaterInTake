# 🚀 DroidStack Beginner Guide (Hindi/Hinglish)

Agar aap developer nahi hain aur is project ko apne Android Studio mein run karna chahte hain, toh ye steps follow karein:

### Step 1: Android Studio Download aur Install Karein
Sabse pehle aapke computer mein **Android Studio** hona chahiye.
1. Is link par jayein: [developer.android.com/studio](https://developer.android.com/studio)
2. **Download Android Studio** button par click karein.
3. Install karte waqt saari settings ko **Default** rehne dein aur "Next" click karte rahein.

### Step 2: Project ko Android Studio mein Clone Karein
Cloning ka matlab hai GitHub se project ki copy apne computer mein laana.
1. Android Studio open karein.
2. Welcome screen par **"Get from VCS"** (ya "Get from Version Control") par click karein.
3. Ek window khulega, wahan **URL** box mein ye link paste karein:
   `https://github.com/ThecodingShef/DroidStack-Template.git`
4. Niche **Clone** button par click karein.
5. Agar login maange toh skip kar sakte hain ya GitHub se login kar sakte hain.

### Step 3: Gradle Sync hone ka intezar karein
1. Project khulne ke baad niche right-side mein **"Gradle Sync"** ya progress bar chalegi.
2. **IMPORTANT:** Jab tak ye khatam na ho jaye, kuch na karein. Ismein 2 se 10 minute lag sakte hain (aapke internet pe depend karta hai).
3. Jab niche "Build Successful" likha aa jaye, tabhi agla step karein.

### Step 4: App ko Run (Chalu) Karein
App dekhne ke liye aapke paas do option hain:

**Option A: Apne real phone par (Best for speed)**
1. Apne Android phone ki **Settings > About Phone** mein jayein.
2. **Build Number** par 7 baar click karein jab tak "You are now a developer" na likha aa jaye.
3. Settings mein **Developer Options** search karein aur **USB Debugging** ko ON karein.
4. Phone ko USB cable se PC se connect karein.
5. Android Studio mein upar phone ka naam dikhega, uske bagal mein **Green Play Button** ▶️ par click karein.

**Option B: Emulator (Virtual Phone) par**
1. Android Studio mein right-side mein **"Device Manager"** par click karein.
2. **"Create Device"** par click karke koi bhi phone (e.g., Pixel 7) select karein aur "Finish" karein.
3. Upar **Green Play Button** ▶️ par click karein.

---

**Kuch Zaruri Baatein (FAQs):**
- **Error aaye toh?** Ek baar `File > Invalidate Caches... > Invalidate and Restart` karein.
- **Internet:** Pehli baar setup karne ke liye internet fast hona chahiye kyunki Android Studio kuch files download karta hai.

🎉 **Bas ho gaya! Aapka DroidStack app chalu ho jayega.**
